Free heatmaps plugin for web analytics, on both PC and smartphone.

== Description ==

UserHeat is free heatmap analytics plugin to visualize user behavior
both PC and smartphone.
It takes just one step and 30 seconds to start analysis.

The key features of the plugin are:


・3 Heatmap(gaze,click,mouse track) reveals see exactly where your
visitors click on the page, see how much attention a specific area
gets by thermography

・Optimize forms usability to improve submission rates.

・It is available not only for PC but also smartphones and tablet devices.


Visit <a href="http://en.userheat.com">userheat</a> to learn more.


== Installation ==

1, Install plugin

2, Go to <a href="http://en.userheat.com">UserHeat</a> website and register URL, e-mail, password.

3, Register the issued site ID.

4, Check report after accumulating site access.
